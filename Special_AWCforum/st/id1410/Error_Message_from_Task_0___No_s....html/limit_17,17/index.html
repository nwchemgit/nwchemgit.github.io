NWCHEM_TOP=/opt/ZIP/nwchem-6.3/
(you can spot this from the line

)

program = /nfs/app1/centos6.3/gnu/apps/NWChem/bin/nwchem

The line should read, instead

program = /opt/ZIP/nwchem-6.3_gnu/bin/LINUX64/nwchem
