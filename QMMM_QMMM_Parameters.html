QM/MM parameters¶

The QM/MM interface parameters define the interaction between classical and quantum regions. 

 qmmm  
 [ [eref](qmmm_eref) <double precision default 0.0d0>]  
 [ [bqzone](qmmm_bq_zone.md) <double precision default 9.0d0>]  
 [ [mm_charges](qmmm_mm_charges.md) [exclude <(none||all||linkbond||linkbond_H) default none>]  
          [ expand  <none||all||solute||solvent> default none]  
          [ update  <integer default 0>]   
 [ [link_atoms](qmmm_link_atoms.md) <(hydrogen||halogen) default hydrogen>]  
 [ [link_ecp](qmmm_link_ecp.md)  <(auto||user) default auto>]  
 [ [region](qmmm_region.md)   < [region1]  [region2]  [region3] > ]  
 [ [method](qmmm_method.md)   [method1]  [method2]  [method3]  ]  
 [ [maxiter](Qmmm_maxiter.md)  [maxiter1] [maxiter2] [maxiter3] ]  
 [ [ncycles](Qmmm_ncycles.md)  < [number] default 1 > ]  
 [ [density](Qmmm_density.md)  [espfit] [static] [dynamical] ]  
 [ [xyz](qmmm_xyz.md)  ]  
 [ [convergence](Qmmm_convergence.md) <double precision default 1.0d-4>] ]  
 [ [load](qmmm_load.md) ]  
 [ [nsamples](Qmmm_nsamples.md) ]  
 end
